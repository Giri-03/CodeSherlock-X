## 📞 TROUBLESHOOTING

**Images not showing?**
1. Verify file names (lowercase, exact match)
2. Check path: `frontend/public/branding/`
3. Restart dev server: `npm run dev`
4. Hard refresh: Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac)

**Build errors?**
1. Ensure PNG files exist in correct location
2. Check file names are exact match
3. Run `npm run build` for validation
4. Clear `.next` folder if needed

**Favicon not updating?**
1. Hard refresh browser cache
2. Check favicon file exists
3. Try different browser to test
4. Wait a few seconds for cache invalidation
